簡易なプライベート記事を作る話。

Nov 14, 2015   #Blog  #Programing  #JavaScript 

動的ページを作れないサーバーで、JavaScriptを使ってURLを知らない人しかアクセスできないようにします。ただし、普通に検索エンジンの検索結果に乗ります。

一次的に見せたいときに置く程度と 聞こえがいいというだけです。

ソースコード

//Formを開くためのボタン
<a onclick="Form();"><i class="fa fa-pencil fa-2x"></i>Open Form</a>

//スクリプト
<script type="text/javascript">
function Form()
{
    var UserInput = prompt("Form:","");
    //入力フォームに記入されていれば(空でなければ)その書かれたところへ飛ぶ
    if( UserInput != null )
    {
            location.href = UserInput;
    }
}
</script>

仕組み

ソースを見ただけでわかると思いますが、ただの子供だましにか過ぎません。

http://localhost:1313/post/ にこのソースを置いた状態でFormを開きHogeと入力すると、http://localhost:1313/post/hoge/に飛びます

それだけです。
