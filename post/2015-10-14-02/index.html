VSCode v0.9.1

Oct 14, 2015   #Blog  #VSCode 

VSCode V0.9.1 10月アップデート 日本語訳まとめ

ここを見るより本文を読んだ方が速いし正確です

拡張機能のインストール先の変更

  • 拡張機能のインストール先がユーザーディレクトリの下になった

  • Windows %USERPROFILE%\.vscode\extensions

  • Mac $HOME/.vscode/extensions

  • Linux $HOME/.vscode/extensions

Yeomanで機能拡張開発

  • プログラミング言語向けのシンタックスハイライトの設定

  • テーマの開発

  • 1.npm install -g yo 2.npm install -g generator-code でジェネレーターのインストール

デバッグコンソールの拡張,改善

  • 視認性の向上

  • コンテキストメニューの拡張

  • 要素の拡張がセッション間で保持される

TypeScript 1.6への対応

{
    "version": "0.1.0",
    "command": "tsc",
    "isShellCommand": true,
    // define the task to be a watching task
    "isWatching": true,
    // use the standard tsc in watch mode problem matcher to find compile problems in the output.
    "problemMatcher": "$tsc-watch"
}

Markdownのプレビュー機能拡張

  • 全体のフォントサイズ,感覚の最適化

  • 画像などのプレビューにも対応(?)

Unity Shader Fileでのシンタックスハイライトのサポート

  • シンタックスハイライトのサポート

新しい設定

  • window.openInNewWindow 新しいファイルを新しいウィンドウで開く or 最後にアクティブになったウィンドウを再利用して開く

  • window.reopenFolders 前回閉じたフォルダの再展開でスタート or 再展開しないでスタート

バグ修正

  • 18998: newly added files not honored by tsconfig.json
  • 18692: Changes to tsconfig.json not updated immediately
  • 19239: Detaching from Unity Results in a Unity Crash
