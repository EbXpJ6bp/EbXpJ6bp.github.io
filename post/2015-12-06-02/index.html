C言語のソースの処理時間を適当にソース測定する

Dec 6, 2015   #Blog  #Programming  #C/C++ 

適当にtime.hを使って適当に測定する

準備

clock()関数を使うため #include <time.h>を使います

ソースコード

#include <stdio.h>
#include <time.h>

int main(void)
{
    clock_t start, end;

    /* 開始 */
    start = clock();

    /* 何らかの処理 */

    int i;
    for (i = 0; i < 10000; i++) {
        printf("%d人目の幼女 ", i);
    }

    /* 終了 */
    end = clock();

    printf("\n=========================\n"
            " 開始時刻:%d\n"
            " 終了時刻:%d\n" 
            " 処理時間:%d[ms]\n", start, end, end - start);

    return 0;

}

実行結果:

=========================

開始時刻:0

終了時刻:931

処理時間:931[ms]
